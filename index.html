🌐 Horizon Local Proxy

A localhost-first browser web proxy for legitimate personal testing, development, and access through a browser you control.

About this project: This is a Node.js-based web proxy that allows you to browse the web through a local proxy server. It's useful for testing, development, and accessing content in controlled environments.

Features

  • ✅ Browse through a localhost proxy
  • ✅ Handles root-relative app navigation
  • ✅ Built-in ad blocking
  • ✅ Session-based cookie management
  • ✅ HTML/CSS/JavaScript rewriting

Quick Start

npm install
npm start

Then open http://127.0.0.1:8080

Environment Variables

  • PORT=3000 - Change the port (default: 8080)
  • HOST=0.0.0.0 - Bind to all interfaces (default: 127.0.0.1)
  • ALLOW_PRIVATE=1 - Allow private network targets
  • DEBUG_PROXY=1 - Enable debug logging
